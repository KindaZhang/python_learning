类的继承：编写类的时候，不一定从空白开始。一个类继承另一个类，就自动获得她的全部属性和方法。
原来的类称为父类又称超类。新创建的类称子类。
除了继承父类以外，还可以拥有自己的属性和方法
子类的__init__（）如下
class ElectricCar(Car):#这是子类的定义，括号内指定父类的名称
    """电动汽车的独特之处"""
    def __init__(self, make, model, year):#这个方法接受创建Car实例所需的信息
        """初始化父类的属性"""
        super().__init__(make, model, year)#这就是为什么称父类为超类的原因
可以看到子类的括号内包含着父类的名称。
父类需要和子类在一个文件中，且要在子类之前
如何重写父类的方法：在子类中写一个同名字的方法就可以了，这样调用子类时采用的就是子类的方法
将一个类作为另一个类的属性：和父类定义是一样的就是在子类里__init__()将这个类赋值给子类的属性
